# BlinkIT Sales Analysis Dashboard (Power BI)

## Project Overview
This Power BI dashboard analyzes BlinkIT sales performance, outlet distribution, item categories, and customer ratings. The dashboard provides interactive insights to help understand sales trends and business performance.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel Dataset

## Key KPIs
- Total Sales: 1.20M
- Average Sales: 140.99
- Number of Items: 9K
- Average Rating: 3.92

## Dashboard Features
- Sales analysis by outlet type
- Sales analysis by outlet size
- Sales analysis by outlet location
- Item type performance analysis
- Fat content distribution
- Outlet establishment trends
- Interactive filters and slicers

## Key Insights
- Tier 3 outlets generated the highest sales (472.13K).
- Low Fat products contributed approximately 65% of total sales.
- Fruits and Vegetables were the top-selling item category.
- Sales peaked around 2018 with over 200K sales.
- Supermarket Type 1 generated the highest overall sales.
